## ğŸ“¦ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or pnpm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/SpiritOfKedar/spirit-works.git
   ```

2. **Navigate to the project**
   ```bash
   cd spirit-works
   ```

3. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

5. **Open in browser**
   
   Visit [http://localhost:3000](http://localhost:3000)
